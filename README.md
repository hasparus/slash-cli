Transform those pesky backslashes into universal forward slashes inside of your shell.

# Install

```sh
npm i -g slash-cli
```

# Usage

```sh
> slash -h
    Usage:
      > slash path\\to\\my\\file
      path/to/my/file
```

# Motivation
